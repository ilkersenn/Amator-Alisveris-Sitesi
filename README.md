# Proje Adı  
Amatör alışveriş sitesi çalışması gibi, kullanıcı girişi, sepete ekleme, mesajlaşma özellikleri var.

## Özellikler  
- Kullanıcı girişi  
- Sepete ürün ekleme  
- Uçak bileti satın alma  
- Mesaj gönderme bölümü  

## Nasıl Çalışır?  
Proje React + Node.js ile geliştirilmiştir.
